# 🐍 Fluffy Snake

カラフルでふわふわな見た目のスネークゲーム 🎨  
ブラウザで遊べる、かわいいシンプルゲーム

---

## 🎮 遊び方
- スタート画面で「時間制限」や「壁のすり抜け（Wrap）」を選択  
- 矢印キー / WASD でスネークを操作  
- スマホの場合はスワイプで操作  
- アイテムを食べるとスコアアップ✨  
- 爆弾（💣）に当たるとゲームオーバー💥  
- 時間制限を過ぎてもゲームオーバー⏰  
- スコアはローカル保存され、ベストスコアが記録されます！

---

## 🍓 アイテムの種類
| 絵文字 | 名前 | 効果 |
|:--:|:--|:--|
| 🍓 | いちご | +1点 |
| 🍒 | さくらんぼ | +2点 |
| 🍎 | りんご | +3点 |
| 🌟 | スター | +5点 |
| 💣 | 爆弾 | 当たると即ゲームオーバー |

---

## ⚙️ 機能
- 時間制限の有無を選択（30s / 60s / 120s / 180s / なし）
- Wrapモード（端から反対側へ出る）
- ローカルハイスコア保存
- PC・スマホ両対応
- 効果音なし
